SkyHelp es un proyecto propio donde apliqué desarrollo frontend y backend en conjunto: interfaz construida con HTML, CSS y JavaScript, conectada a lógica de aplicación en C#. Fue mi manera de pasar del diseño visual a construir software funcional de principio a fin.
